# 🎨 Kulveer Singh — Developer Portfolio

A personal portfolio website with an IDE/Terminal-inspired theme, showcasing my work, skills, and experience as a **Full-Stack & Web Developer**.

**🔗 Live Demo:** [kulveersingh60.github.io/portfolio](https://kulveersingh60.github.io/portfolio)

---

## ✨ Features

### Design & UX
- 🎨 **IDE/Terminal Aesthetic** - VS Code & JetBrains Mono inspired design
- 📱 **Fully Responsive** - Mobile, tablet, and desktop optimized
- ✨ **Smooth Animations** - Typing effects, scroll animations, 3D tilt cards
- 🎯 **Modern UI** - Gradient background, syntax-highlighted code windows
- 🌙 **Professional Layout** - Status bar, organized sections, clean typography

### Content Sections
- 👋 **Hero Section** - Animated greeting with typing effect
- 💼 **About Me** - Brief introduction and focus areas
- 🎯 **Skills** - Interactive tabs showcasing expertise:
  - Core Development (PHP, MySQL, JavaScript, HTML/CSS)
  - Databases & Tools (MySQL, Git, VS Code, Bootstrap)
  - Business & Office (Project Management, Communication)
  - Digital Marketing (SEO, Content Strategy)
  - Soft Skills (Problem Solving, Teamwork, Leadership)
- 🚀 **Projects** - Featured work with links and descriptions
- 💼 **Experience** - Professional background and roles
- 🏆 **Certifications** - Achievements and credentials
- 📫 **Contact** - Multiple ways to reach out

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for cloning)

### Installation

**Option 1: Direct Download**
1. Download the repository as ZIP
2. Extract to your desired location
3. Open `index.html` in your browser

**Option 2: Clone with Git**
```bash
git clone https://github.com/KulveerSingh60/portfolio.git
cd portfolio
# Open index.html in your browser
```

**Option 3: Live Demo**
Visit: [kulveersingh60.github.io/portfolio](https://kulveersingh60.github.io/portfolio)

## 🎨 Customization

### Change Personal Information
Edit `index.html` and update:
- Your name and title
- Contact email
- Social media links
- Project descriptions

### Customize Colors
Edit `css/style.css`:
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  --accent-color: #your-color;
}
```

### Modify Content
- Update skills in the skills section
- Add/remove projects
- Change certifications
- Update experience
