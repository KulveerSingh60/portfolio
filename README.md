# Kulveer Singh — Developer Portfolio

A personal portfolio website with an IDE / terminal-inspired theme, built to showcase my
work, skills, and experience as a full-stack & web developer.

🔗 **Live site:** _add your GitHub Pages / hosting URL here_

## ✨ Features

- Responsive, mobile-friendly layout (Bootstrap 5)
- Coder / IDE aesthetic — JetBrains Mono, syntax-highlighted code windows, VS Code–style status bar
- Animated hero with a typing effect
- Interactive skills tabs (Core Dev, Databases & Tools, Business & Office, Digital Marketing, Soft Skills)
- Projects, experience, and certifications sections
- 3D tilt cards, scroll animations (AOS), and animated gradient background

## 🛠️ Built With

- **HTML5** & **CSS3**
- **Bootstrap 5**
- **JavaScript** (vanilla)
- [Bootstrap Icons](https://icons.getbootstrap.com/), [AOS](https://michalsnik.github.io/aos/), Google Fonts

## 📁 Structure

```
portfolio/
```

## 🚀 Run Locally

Just open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/<your-username>/portfolio.git
cd portfolio
# then open index.html
```
